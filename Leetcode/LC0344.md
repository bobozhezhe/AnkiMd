# Leetcode
## 0344 Reverse String
### Question
Write a function that reverses a string. The input string is given as an array of characters s.

You must do this by modifying the input array in-place with O(1) extra memory.

### Example 1:

Input: s = ["h","e","l","l","o"]
Output: ["o","l","l","e","h"]
Example 2:

Input: s = ["H","a","n","n","a","h"]
Output: ["h","a","n","n","a","H"]

### Solution 1
```java
class Solution {
    public void reverseString(char[] s) {
        int len = s.length;
        if(len <= 1) return;

        int high = len - 1, low = 0;
        while (low < high) {
            char tmp = s[high];
            s[high] = s[low];
            s[low] = tmp;
            ++low;
            --high;
        }
    }
}
```
